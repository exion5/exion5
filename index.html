About

My interest in hardware started as a kid when I broke my Nintendo Switch. After getting it repaired by an Electrical Engineer, I became fascinated by how such a small device could hold so many systems that work together. It sparked my curiosity about how electronic devices are designed at each level. It pushed me to start learning more about circuits and acted as my first stepping stone in becoming an Electrical Engineer.

Since then I've designed many projects such as creating a mechanical keyboard, Raspberry Pi handheld console, and an IEEE Technical Design award-winning environmental monitoring system integrating mmWave radar, particulate sensors, and a live web dashboard on a Pi Zero 2W.

Most recently I took a full custom PCB from concept to fabrication for my RC Drone project, designing an Arduino Nano equivalent with integrated boost converter, LiPo charger, USB bridge, and battery protection in KiCad, fabricated with JLCPCB.

SCOPE Monitor ★ IEEE Technical Award

SCOPE · Environmental & Security Monitoring System

SCOPE is a Raspberry Pi Zero 2W-based system with a live Flask web dashboard integrating a 60 GHz mmWave radar for detecting movement alongside decibel, particle concentration, and air quality sensors. The toughest part was the radar: it needed a custom binary frame parser for a AA FF 03 00 header format over a bit-banged software UART running at 256 000 baud via pigpio. This was due to a hardware UART-pin conflict with the PMS5003 particulate sensor on /dev/serial0.

RC Drone & Controller Transmitter

2.4 GHz Radio-Controlled Drone & Transmitter

This project involved building a handheld RC transmitter and drone. Both utilize an Arduino Nano as the main brain, and uses NRF24L01 for 2.4 GHz RF. The transmitter utilizes an MPU-6050 for orientation, dual analog joysticks, rotary encoder, and a PCF8574-driven I²C LCD display for proper usage. The drone contains an MPU-6050 for stability / orientation, and uses 716 coreless motors with 55000 RPM. Power delivery is done through a boost converter from a 3.7V LiPo battery.

Custom ATMega328P Board

ATmega328P-MU Board · KiCad to JLCPCB

A full Arduino Nano equivalent designed from scratch in KiCad and fabricated at JLCPCB. Its intended use is for my RC Drone project as a custom flight controller board that can be used wirelessly, so it includes an ATmega328P MCU, FT232RL USB-UART bridge, MCP73831 LiPo charger, DW01A + FS8205A battery protection, and a TPS61022 boost converter for 5V output with feedback resistors calculated from the 0.6V reference: R10 = 750 kΩ, R11 = 100 kΩ. 4-layer layout with 0.8 mm VBUS & power traces, 0.2 mm signal traces, and equal-length USB D+/D− routing.
