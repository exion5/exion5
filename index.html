Projects

01

SCOPE Monitor


★ IEEE Technical Award

An environmental and security monitoring system built on a Raspberry Pi Zero 2W with a live Flask web dashboard. Integrates a 24 GHz mmWave radar for presence detection alongside particulate and air quality sensors. Key engineering challenges included writing a custom binary frame parser for the radar's UART protocol and implementing bit-banged software UART at 256 kbaud via pigpio — hardware UART was already occupied.

Raspberry Pi Flask UART / I²C mmWave Radar Python systemd ADS1115

02

Custom RC Transmitter

A handheld 2.4 GHz radio transmitter built from scratch targeting a custom drone receiver. Designed around an Arduino Nano with an NRF24L01+LNA transceiver, MPU-6050 IMU for orientation, dual analog joysticks, rotary encoder, and a PCF8574-driven I²C LCD. Power delivery uses an AMS1117-3.3V LDO and a BSS138-based logic-level shifter to safely interface the 5V Arduino with 3.3V peripherals.

Arduino Nano NRF24L01+LNA MPU-6050 I²C SPI Power Design Perfboard

03

Custom Arduino PCB

A full Arduino Nano equivalent designed from scratch in KiCad and fabricated at JLCPCB. Integrates an ATmega328P MCU, FT232RL USB-UART bridge, MCP73831 LiPo charger, DW01A + FS8205A battery protection, TPS61022 boost converter (5V output with calculated feedback divider), and AMS1117-3.3 LDO — all on a 2-layer board with properly sized power traces, decoupling caps, and equal-length USB D+/D- routing.

KiCad ATmega328P FT232RL TPS61022 LiPo Charging PCB Layout JLCPCB
