J1 · SCOPE Monitor ★ IEEE Technical Award Sept. 2025 – Mar. 2026

IoT SCOPE Monitor System

    SCOPE is a Raspberry Pi Zero 2W-based system with a live Flask web dashboard integrating a 60 GHz mmWave radar for detecting movement alongside decibel, particle concentration, and air quality sensors. The toughest part was the radar: it needed a custom binary frame parser for a AA FF 03 00 header format over a bit-banged software UART running at 256 000 baud via pigpio. This was because the TX and RX pins were occupied by the PMS5003 particulate sensor on /dev/serial0.

J2 · Custom ATMega328P Board Mar. 2026 – May 2026

ATmega328P PCB

    A full ATMega328P Microcontroller board designed from scratch in KiCad and fabricated at JLCPCB. Its intended use is for my RC Drone project as a custom flight controller board that can be used wirelessly, so it includes an ATmega328P MCU, FT232RL USB-UART bridge, MCP73831 LiPo charger, DW01A + FS8205A battery protection, and a TPS61022 boost converter for 5V output with feedback resistors calculated from the 0.6V reference: R10 = 750 kΩ, R11 = 100 kΩ. 4-layer layout with 0.8 mm VBUS & power traces, 0.2 mm signal traces, and equal-length USB D+/D− routing.

J3 · RC Drone & Transmitter Apr. 2026 – Aug. 2026

2.4 GHz Radio-Controlled Drone & Transmitter

    This project involved building a handheld RC transmitter and drone. Both utilize an Arduino Nano as the main brain, and uses NRF24L01 for 2.4 GHz RF. The transmitter utilizes an MPU-6050 for orientation, dual analog joysticks, rotary encoder, and a PCF8574-driven I²C LCD display for proper usage. The drone contains an MPU-6050 for stability / orientation, and uses 716 coreless motors with 55000 RPM. Power delivery is done through a boost converter from a 3.7V LiPo battery.

J4 · Mechanical Keyboard Mar. 2026 – Apr. 2026

Handmade 68-Key Mechanical Keyboard

    I designed and built a custom 68-key mechanical keyboard from scratch, starting with a hand-drawn layout and switch placement. I used the Teensy microcontroller as the brain, and implemented a diode matrix for accurate key scanning. The firmware was built using QMK, which I configured to handle USB HID communication and custom key mappings. For the case, I designed a sleek enclosure in Fusion 360 and 3D-printed it, then hand-wired all the switch connections to the Teensy.

J5 · Pi Gaming Console Dec. 2024 – Feb. 2025

Raspberry Pi Handheld Gaming Console

    This project involved creating a portable handheld gaming console using a Raspberry Pi 4 and an Arduino Mini for controller input. I set up RetroPie on the Pi for emulation, and designed custom 3D printed stability enclosures on Fusion 360 to house the components. The controller interface was built using a matrix of buttons connected to the Arduino, which communicated with the Pi over USB serial. I also integrated an IPS display and configured a PiSugar battery board for portable power management.

SW1 · Keyboard Firmware Editor Aug. 2026 – Sept. 2026

Keyboard Firmware Editor

    I developed a web app that lets users design a custom keyboard layout and generates compatible firmware. The app features an interactive key mapping interface where users can assign keycodes, macros, and layers to each key. I designed it because I felt that there were too many different websites I used when designing my own mechanical keyboard and wanted a more streamlined experience. The frontend was built with React and TypeScript, and the backend uses Node.js to compile the firmware based on user input.
